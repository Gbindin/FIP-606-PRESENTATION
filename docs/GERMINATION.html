#GRAPHICAL REPRESENTATION

dat3 <- dat |>
  group_by(VARIETY, SOWDATE)|>
summarise(mean = mean(GERM), sd = sd(GERM))

dat$letter = c()
library(scales)
dat3$letter = c("c","b","a","a","a","a","b","b","a","b","b",
                "a","a","a","a")
dat3 |>
  ggplot(aes(SOWDATE, mean))+
  geom_col()+
  facet_wrap(~VARIETY)+
  geom_errorbar(aes(ymin = mean - sd, ymax = mean + sd), 
                width = 0.1)+
  geom_text(aes(label = letter), 
            position = position_stack(vjust = 1.5),
                                      color = "black", 
                                      size = 4)

dat$letters = c()
library(scales)

RESULT: the analysis showed that there was a significant difference (P˂0.001)in the germination percentage of the varieties, with Tvx 3236 having the highest germination percentage while lowest germination percentage was recorded in Ife Brown. Likewise, there was significant difference (P˂0.001) among the three sowing dates. Also, there was significant interaction between sowing date and variety.

The normality and homoscedasticity of the dataset analyzed indicated that both normality and homoscedasticity displayed an even distribution. Based on this, the dataset exhibited an evenly distributed normality and homoscedasticity.
